# Alterações desta versão

- Login mais estreito e mais alto.
- Menu lateral retrátil, com estado salvo no navegador.
- Clique em qualquer cartão da área **OS por etapa** para editar a OS em um painel lateral.
- Arraste cartões entre etapas para mudar o status.
- Arraste cartões para cima ou para baixo na mesma etapa para alterar a prioridade de desenvolvimento. A posição aparece no canto do cartão.
- Backlog dos indicadores redesenhado em cartões compactos de desenvolvedores lado a lado.
- Etapas do backlog expansíveis para reduzir a largura e a poluição visual.

## Uso rápido

1. Abra `index.html`.
2. Entre com as mesmas credenciais já definidas no projeto.
3. Use a seta circular na lateral do menu para recolher ou expandir.
4. No dashboard, clique ou arraste os cartões de OS.
5. Abra `indicadores.html` para a nova visão do backlog.
